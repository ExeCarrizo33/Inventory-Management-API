# Inventory Management API

API RESTful para la gestión de inventarios, desarrollada con Java y Spring Boot.

## Tecnologías utilizadas

- **Java 17** - Lenguaje principal.
- **Spring Boot** - Framework para aplicaciones backend.
- **Spring Data JPA** - Persistencia de datos con bases de datos SQL.
- **H2 Database** - Base de datos en memoria para pruebas.
- **Maven** - Gestión de dependencias y construcción del proyecto.
